# A Quasar Project

https://quasar-project-4ktkpk721-jorens-projects-e792a21c.vercel.app

## Install the dependencies
```bash
npm install
```

### Start the app in web/SPA
```bash
quasar dev
```

### Start the app in desktop mode
```bash
quasar dev -m electron
```
### Start the app in PWA
```bash
quasar dev -m pwa
```
